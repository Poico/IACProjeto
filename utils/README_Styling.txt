To use the Notepad++ styling for PEPE16:
1) Open Settings>Style Configurator...
2) Select theme 'Black board'
3) Open Language>User Defined Language>Define your language...
4) Click 'Import'
5) Navigate and select the 'Styling.xml' file
6) Restart Notepad++
7) Select Language>PEPE16
